# Exile [POST]

```/api/ranking/exile/```

This endpoint exiled the specified user.

## Request Body

| Body      | Description           | Type              |
| ----------- | ---------------------|--------------- |
| `userId`       | The user who you want to exile | `INTEGER` |
| `groupId`    | The group which the user is in | `INTEGER` | 
| `token`    | Exile token | `STRING` |
